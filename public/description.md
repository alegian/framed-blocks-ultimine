# Features
The main feature of this mod is that you can apply camo to many framed blocks at once, simply right clicking with ultimine. The mod will simulate even rotated blocks, so split framed blocks should properly get their camo as if right clicked in the same angle.

That being said, any sort of right click should work, like glowstone or wrench or other.

# Motivation
This mod was mainly created as a quality of life addition to my personal Star Technology modpack playthrough. It would be awesome if it were included in the original modpack too sometime.

# Porting to newer versions
This mod is intended to remain at MC 1.20 forever. If you want the feature for newer versions, the right thing to do is to submit a PR to the original Framed Blocks mod. The only reason I didn't do that is that the author clearly said they ended support for the 1.20 branch, apart from breaking changes.
